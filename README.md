# OS Project – Intelligent Real-Time File Compression System

## Overview

An operating systems project that monitors folders in real time and automatically compresses files using adaptive compression techniques.

The system integrates file monitoring, metadata tracking, compression scheduling, analytics, and machine-learning-based decision making to optimize storage utilization while maintaining performance.

## Features

* Real-time folder monitoring
* Automatic file compression
* Huffman encoding support
* Compression queue management
* Metadata tracking
* Compression analytics dashboard
* Machine learning based compression prediction
* Web-based user interface
* Compression statistics and reports
